46  /**
47   * <p>
48   * XWorkConverter is a singleton used by many of the Struts 2's Ognl extension points,
49   * such as InstantiatingNullHandler, XWorkListPropertyAccessor etc to do object
50   * conversion.
51   * </p>
52   *
53   * <!-- START SNIPPET: javadoc -->
54   *
55   * <p>
56   * Type conversion is great for situations where you need to turn a String in to a more complex object. Because the web
57   * is type-agnostic (everything is a string in HTTP), Struts 2's type conversion features are very useful. For instance,
58   * if you were prompting a user to enter in coordinates in the form of a string (such as "3, 22"), you could have
59   * Struts 2 do the conversion both from String to Point and from Point to String.
60   * </p>
61   *
62   * <p>
63   * Using this "point" example, if your action (or another compound object in which you are setting properties on)
64   * has a corresponding ClassName-conversion.properties file, Struts 2 will use the configured type converters for
65   * conversion to and from strings. So turning "3, 22" in to new Point(3, 22) is done by merely adding the following
66   * entry to <b>ClassName-conversion.properties</b> (Note that the PointConverter should impl the TypeConverter
67   * interface):
68   * </p>
69   * <p><b>point = com.acme.PointConverter</b></p>
70   *
71   * <p>
72   * Your type converter should be sure to check what class type it is being requested to convert. Because it is used
73   * for both to and from strings, you will need to split the conversion method in to two parts: one that turns Strings in
74   * to Points, and one that turns Points in to Strings.
75   * </p>
76   *
77   * <p>
78   * After this is done, you can now reference your point (using &lt;s:property value="point"/&gt; in JSP or ${point}
79   * in FreeMarker) and it will be printed as "3, 22" again. As such, if you submit this back to an action, it will be
80   * converted back to a Point once again.
81   * </p>
82   *
83   * <p>
84   * In some situations you may wish to apply a type converter globally. This can be done by editing the file
85   * <b>xwork-conversion.properties</b> in the root of your class path (typically WEB-INF/classes) and providing a
86   * property in the form of the class name of the object you wish to convert on the left hand side and the class name of
87   * the type converter on the right hand side. For example, providing a type converter for all Point objects would mean
88   * adding the following entry:
89   * </p>
90   *
91   * <p><b>com.acme.Point = com.acme.PointConverter</b></p>
92   *
93   * <!-- END SNIPPET: javadoc -->
94   *
95   *
96   * <!-- START SNIPPET: i18n-note -->
97   * <p>
98   * Type conversion should not be used as a substitute for i18n. It is not recommended to use this feature to print out
99   * properly formatted dates. Rather, you should use the i18n features of Struts 2 (and consult the JavaDocs for JDK's
100  * MessageFormat object) to see how a properly formatted date should be displayed.
101  * </p>
102  * <!-- END SNIPPET: i18n-note -->
103  *
104  *
105  * <!-- START SNIPPET: error-reporting -->
106  * <p>
107  * Any error that occurs during type conversion may or may not wish to be reported. For example, reporting that the
108  * input &quot;abc&quot; could not be converted to a number might be important. On the other hand, reporting that an empty string,
109  * &quot;&quot;, cannot be converted to a number might not be important - especially in a web environment where it is hard to
110  * distinguish between a user not entering a value vs. entering a blank value.
111  * </p>
112  * <p> By default, all conversion errors are reported using the generic i18n key <b>xwork.default.invalid.fieldvalue</b>,
113  * which you can override (the default text is <i>Invalid field value for field "xxx"</i>, where xxx is the field name)
114  * in your global i18n resource bundle.
115  * </p>
116  *
117  * <p>However, sometimes you may wish to override this message on a per-field basis. You can do this by adding an i18n
118  * key associated with just your action (Action.properties) using the pattern <b>invalid.fieldvalue.xxx</b>, where xxx
119  * is the field name.
120  * </p>
121  *
122  * <p>It is important to know that none of these errors are actually reported directly. Rather, they are added to a map
123  * called <i>conversionErrors</i> in the ActionContext. There are several ways this map can then be accessed and the
124  * errors can be reported accordingly.
125  * </p>
126  * <!-- END SNIPPET: error-reporting -->

193     public static String getConversionErrorMessage(String propertyName, Class toClass, ValueStack stack) {
194         LocalizedTextProvider localizedTextProvider = ActionContext.getContext().getContainer().getInstance(LocalizedTextProvider.class);
195         String defaultMessage = localizedTextProvider.findDefaultText("xwork.default.invalid.fieldvalue",
196                 ActionContext.getContext().getLocale(),
197                 new Object[]{
198                         propertyName
199                 });
200 
201         List<String> indexValues = getIndexValues(propertyName);
202 
203         propertyName = removeAllIndexesInPropertyName(propertyName);
204 
205         String prefixedPropertyName = CONVERSION_ERROR_PROPERTY_PREFIX + propertyName;
206         String getTextExpression = "getText('" + prefixedPropertyName + "')";
207         String message = (String) stack.findValue(getTextExpression);
208 
209         if (message == null || prefixedPropertyName.equals(message)) {
210             getTextExpression = "getText('" + CONVERSION_ERROR_PROPERTY_PREFIX + toClass.getName() + "','" + defaultMessage + "')";
211             message = (String) stack.findValue(getTextExpression);
212         }
213 
214         if (message == null) {
215             message = defaultMessage;
216         } else {
217             message = MessageFormat.format(message, indexValues.toArray());
218         }
219 
220         return message;
221     }
222 
223     private static String removeAllIndexesInPropertyName(String propertyName) {
224         return propertyName.replaceAll(MESSAGE_INDEX_PATTERN, PERIOD);
225     }
226 
227     private static List<String> getIndexValues(String propertyName) {
228         Matcher matcher = messageIndexPattern.matcher(propertyName);
229         List<String> indexes = new ArrayList<>();
230         while (matcher.find()) {
231             Integer index = new Integer(matcher.group().replaceAll(MESSAGE_INDEX_BRACKET_PATTERN, "")) + 1;
232             indexes.add(Integer.toString(index));
233         }
234         return indexes;
235     }

247     /**
248      * Convert value from one form to another.
249      * Minimum requirement of arguments:
250      * <ul>
251      * <li>supplying context, toClass and value</li>
252      * <li>supplying context, target and value.</li>
253      * </ul>
254      *
255      * @see TypeConverter#convertValue(java.util.Map, java.lang.Object, java.lang.reflect.Member, java.lang.String, java.lang.Object, java.lang.Class)
256      */
257     @Override
258     public Object convertValue(Map<String, Object> context, Object target, Member member, String property, Object value, Class toClass) {
259         //
260         // Process the conversion using the default mappings, if one exists
261         //
262         TypeConverter tc = null;
263 
264         if ((value != null) && (toClass == value.getClass())) {
265             return value;
266         }
267 
268         // allow this method to be called without any context
269         // i.e. it can be called with as little as "Object value" and "Class toClass"
270         if (target != null) {
271             Class clazz = target.getClass();
272 
273             Object[] classProp = null;
274 
275             // this is to handle weird issues with setValue with a different type
276             if ((target instanceof CompoundRoot) && (context != null)) {
277                 classProp = getClassProperty(context);
278             }
279 
280             if (classProp != null) {
281                 clazz = (Class) classProp[0];
282                 property = (String) classProp[1];
283             }
284 
285             tc = (TypeConverter) getConverter(clazz, property);
286             LOG.debug("field-level type converter for property [{}] = {}", property, (tc == null ? "none found" : tc));
287         }
288 
289         if (tc == null && context != null) {
290             // ok, let's see if we can look it up by path as requested in XW-297
291             Object lastPropertyPath = context.get(ReflectionContextState.CURRENT_PROPERTY_PATH);
292             Class clazz = (Class) context.get(XWorkConverter.LAST_BEAN_CLASS_ACCESSED);
293             if (lastPropertyPath != null && clazz != null) {
294                 String path = lastPropertyPath + "." + property;
295                 tc = (TypeConverter) getConverter(clazz, path);
296             }
297         }
298 
299         if (tc == null) {
300             if (toClass.equals(String.class) && (value != null) && !(value.getClass().equals(String.class) || value.getClass().equals(String[].class))) {
301                 // when converting to a string, use the source target's class's converter
302                 tc = lookup(value.getClass());
303             } else {
304                 // when converting from a string, use the toClass's converter
305                 tc = lookup(toClass);
306             }
307 
308             if (LOG.isDebugEnabled())
309                 LOG.debug("global-level type converter for property [{}] = {} ", property, (tc == null ? "none found" : tc));
310         }
311 
312 
313         if (tc != null) {
314             try {
315                 return tc.convertValue(context, target, member, property, value, toClass);
316             } catch (Exception e) {
317                 LOG.debug("Unable to convert value using type converter [{}]", tc.getClass().getName(), e);
318                 handleConversionException(context, property, value, target, toClass);
319 
320                 return TypeConverter.NO_CONVERSION_POSSIBLE;
321             }
322         }
323 
324         if (defaultTypeConverter != null) {
325             try {
326                 LOG.debug("Falling back to default type converter [{}]", defaultTypeConverter);
327                 return defaultTypeConverter.convertValue(context, target, member, property, value, toClass);
328             } catch (Exception e) {
329                 LOG.debug("Unable to convert value using type converter [{}]", defaultTypeConverter.getClass().getName(), e);
330                 handleConversionException(context, property, value, target, toClass);
331 
332                 return TypeConverter.NO_CONVERSION_POSSIBLE;
333             }
334         } else {
335             try {
336                 LOG.debug("Falling back to Ognl's default type conversion");
337                 return super.convertValue(value, toClass);
338             } catch (Exception e) {
339                 LOG.debug("Unable to convert value using type converter [{}]", super.getClass().getName(), e);
340                 handleConversionException(context, property, value, target, toClass);
341 
342                 return TypeConverter.NO_CONVERSION_POSSIBLE;
343             }
344         }
345     }

347     /**
348      * Looks for a TypeConverter in the default mappings.
349      *
350      * @param className name of the class the TypeConverter must handle
351      * @param isPrimitive is primitive?
352      * @return a TypeConverter to handle the specified class or null if none can be found
353      */
354     public TypeConverter lookup(String className, boolean isPrimitive) {
355         if (converterHolder.containsUnknownMapping(className) && !converterHolder.containsDefaultMapping(className)) {
356             return null;
357         }
358 
359         TypeConverter result = converterHolder.getDefaultMapping(className);
360 
361         //Looks for super classes
362         if (result == null && !isPrimitive) {
363             Class clazz = null;
364 
365             try {
366                 clazz = Thread.currentThread().getContextClassLoader().loadClass(className);
367             } catch (ClassNotFoundException cnfe) {
368                 LOG.debug("Cannot load class {}", className, cnfe);
369             }
370 
371             result = lookupSuper(clazz);
372 
373             if (result != null) {
374                 //Register now, the next lookup will be faster
375                 registerConverter(className, result);
376             } else {
377                 // if it isn't found, never look again (also faster)
378                 registerConverterNotFound(className);
379             }
380         }
381 
382         return result;
383     }

385     /**
386      * Looks for a TypeConverter in the default mappings.
387      *
388      * @param clazz the class the TypeConverter must handle
389      * @return a TypeConverter to handle the specified class or null if none can be found
390      */
391     public TypeConverter lookup(Class clazz) {
392         TypeConverter result = lookup(clazz.getName(), clazz.isPrimitive());
393 
394         if (result == null && clazz.isPrimitive()) {
395             /**
396              * if it is primitive use default converter which allows to define different converters per type
397              * @see XWorkBasicConverter
398              */
399             return defaultTypeConverter;
400         }
401 
402         return result;
403     }

405     protected Object getConverter(Class clazz, String property) {
406         LOG.debug("Retrieving convert for class [{}] and property [{}]", clazz, property);
407 
408         synchronized (clazz) {
409             if ((property != null) && !converterHolder.containsNoMapping(clazz)) {
410                 try {
411                     Map<String, Object> mapping = converterHolder.getMapping(clazz);
412 
413                     if (mapping == null) {
414                         mapping = buildConverterMapping(clazz);
415                     } else {
416                         mapping = conditionalReload(clazz, mapping);
417                     }
418 
419                     Object converter = mapping.get(property);
420                     if (converter == null && LOG.isDebugEnabled()) {
421                         LOG.debug("Converter is null for property [{}]. Mapping size [{}]:", property, mapping.size());
422                         for (Map.Entry<String, Object> entry : mapping.entrySet()) {
423                             LOG.debug("{}:{}", entry.getKey(), entry.getValue());
424                         }
425                     }
426                     return converter;
427                 } catch (Throwable t) {
428                     LOG.debug("Got exception trying to resolve convert for class [{}] and property [{}]", clazz, property, t);
429                     converterHolder.addNoMapping(clazz);
430                 }
431             }
432         }
433         return null;
434     }

436     protected void handleConversionException(Map<String, Object> context, String property, Object value, Object object, Class toClass) {
437         if (context != null && (Boolean.TRUE.equals(context.get(REPORT_CONVERSION_ERRORS)))) {
438             String realProperty = property;
439             String fullName = (String) context.get(CONVERSION_PROPERTY_FULLNAME);
440 
441             if (fullName != null) {
442                 realProperty = fullName;
443             }
444 
445             Map<String, ConversionData> conversionErrors = (Map<String, ConversionData>) context.get(ActionContext.CONVERSION_ERRORS);
446 
447             if (conversionErrors == null) {
448                 conversionErrors = new HashMap<>();
449                 context.put(ActionContext.CONVERSION_ERRORS, conversionErrors);
450             }
451 
452             conversionErrors.put(realProperty, new ConversionData(value, toClass));
453         }
454     }

464     private Object[] getClassProperty(Map<String, Object> context) {
465         Object lastClass = context.get(LAST_BEAN_CLASS_ACCESSED);
466         Object lastProperty = context.get(LAST_BEAN_PROPERTY_ACCESSED);
467         return (lastClass != null && lastProperty != null) ? new Object[] {lastClass, lastProperty} : null;

470     /**
471      * Looks for converter mappings for the specified class and adds it to an existing map.  Only new converters are
472      * added.  If a converter is defined on a key that already exists, the converter is ignored.
473      *
474      * @param mapping an existing map to add new converter mappings to
475      * @param clazz   class to look for converter mappings for
476      */
477     protected void addConverterMapping(Map<String, Object> mapping, Class clazz) {
478         // Process <clazz>-conversion.properties file
479         String converterFilename = buildConverterFilename(clazz);
480         fileProcessor.process(mapping, clazz, converterFilename);
481 
482         // Process annotations
483         Annotation[] annotations = clazz.getAnnotations();
484 
485         for (Annotation annotation : annotations) {
486             if (annotation instanceof Conversion) {
487                 Conversion conversion = (Conversion) annotation;
488                 for (TypeConversion tc : conversion.conversions()) {
489                     if (mapping.containsKey(tc.key())) {
490                         break;
491                     }
492                     if (LOG.isDebugEnabled()) {
493                         if (StringUtils.isEmpty(tc.key())) {
494                             LOG.debug("WARNING! key of @TypeConversion [{}/{}] applied to [{}] is empty!", tc.converter(), tc.converterClass(), clazz.getName());
495                         } else {
496                             LOG.debug("TypeConversion [{}/{}] with key: [{}]", tc.converter(), tc.converterClass(), tc.key());
497                         }
498                     }
499                     annotationProcessor.process(mapping, tc, tc.key());
500                 }
501             }
502         }
503 
504         // Process annotated methods
505         for (Method method : clazz.getMethods()) {
506             annotations = method.getAnnotations();
507             for (Annotation annotation : annotations) {
508                 if (annotation instanceof TypeConversion) {
509                     TypeConversion tc = (TypeConversion) annotation;
510                     String key = tc.key();
511                     // Default to the property name with prefix
512                     if (StringUtils.isEmpty(key)) {
513                         key = AnnotationUtils.resolvePropertyName(method);
514                         switch (tc.rule()) {
515                             case COLLECTION:
516                                 key = DefaultObjectTypeDeterminer.DEPRECATED_ELEMENT_PREFIX + key;
517                                 break;
518                             case CREATE_IF_NULL:
519                                 key = DefaultObjectTypeDeterminer.CREATE_IF_NULL_PREFIX + key;
520                                 break;
521                             case ELEMENT:
522                                 key = DefaultObjectTypeDeterminer.ELEMENT_PREFIX + key;
523                                 break;
524                             case KEY:
525                                 key = DefaultObjectTypeDeterminer.KEY_PREFIX + key;
526                                 break;
527                             case KEY_PROPERTY:
528                                 key = DefaultObjectTypeDeterminer.KEY_PROPERTY_PREFIX + key;
529                                 break;
530                         }
531                         LOG.debug("Retrieved key [{}] from method name [{}]", key, method.getName());
532                     }
533                     if (mapping.containsKey(key)) {
534                         break;
535                     }
536                     annotationProcessor.process(mapping, tc, key);
537                 }
538             }
539         }
540     }

543     /**
544      * Looks for converter mappings for the specified class, traversing up its class hierarchy and interfaces and adding
545      * any additional mappings it may find.  Mappings lower in the hierarchy have priority over those higher in the
546      * hierarchy.
547      *
548      * @param clazz the class to look for converter mappings for
549      * @return the converter mappings
550      * @throws Exception in case of any errors
551      */
552     protected Map<String, Object> buildConverterMapping(Class clazz) throws Exception {
553         Map<String, Object> mapping = new HashMap<>();
554 
555         // check for conversion mapping associated with super classes and any implemented interfaces
556         Class curClazz = clazz;
557 
558         while (!curClazz.equals(Object.class)) {
559             // add current class' mappings
560             addConverterMapping(mapping, curClazz);
561 
562             // check interfaces' mappings
563             Class[] interfaces = curClazz.getInterfaces();
564 
565             for (Class anInterface : interfaces) {
566                 addConverterMapping(mapping, anInterface);
567             }
568 
569             curClazz = curClazz.getSuperclass();
570         }
571 
572         if (mapping.size() > 0) {
573             converterHolder.addMapping(clazz, mapping);
574         } else {
575             converterHolder.addNoMapping(clazz);
576         }
577 
578         return mapping;
579     }

581     private Map<String, Object> conditionalReload(Class clazz, Map<String, Object> oldValues) throws Exception {
582         Map<String, Object> mapping = oldValues;
583 
584         if (reloadingConfigs) {
585             URL fileUrl = ClassLoaderUtil.getResource(buildConverterFilename(clazz), clazz);
586             if (fileManager.fileNeedsReloading(fileUrl)) {
587                 mapping = buildConverterMapping(clazz);
588             }
589         }
590 
591         return mapping;
592     }

594     /**
595      * Recurses through a class' interfaces and class hierarchy looking for a TypeConverter in the default mapping that
596      * can handle the specified class.
597      *
598      * @param clazz the class the TypeConverter must handle
599      * @return a TypeConverter to handle the specified class or null if none can be found
600      */
601     TypeConverter lookupSuper(Class clazz) {
602         TypeConverter result = null;
603 
604         if (clazz != null) {
605             result = converterHolder.getDefaultMapping(clazz.getName());
606 
607             if (result == null) {
608                 // Looks for direct interfaces (depth = 1 )
609                 Class[] interfaces = clazz.getInterfaces();
610 
611                 for (Class anInterface : interfaces) {
612                     if (converterHolder.containsDefaultMapping(anInterface.getName())) {
613                         result = converterHolder.getDefaultMapping(anInterface.getName());
614                         break;
615                     }
616                 }
617 
618                 if (result == null) {
619                     // Looks for the superclass
620                     // If 'clazz' is the Object class, an interface, a primitive type or void then clazz.getSuperClass() returns null
621                     result = lookupSuper(clazz.getSuperclass());
622                 }
623             }
624         }
625 
626         return result;
627     }
